# TSS AUDITOR DE SEGURIDAD

## STACK

Voy a utilizar C++23 con modules. Para compilar el programa usare CMAKE, VisualStudio y el compilador cl.
Utilizare Visual Studio Community 2026 para IDE.
Para la ui utilizare Imgui-docking con Opengl y GLFW.
Para la aplicación cli utilizare CLI11.
Utilizare mi propia herramienta core para prototipar el projecto y despues la quitare.

## APLICACIÓN UI

Con Imgui-docking crear una pantalla con widgets que pueden ser reposicionadas y
cambiadas de tamaño.
